# Aplikacja do zarządzania kadrami

Jest to aplikacja stworzona na potrzeby napisania pracy inżynierskiej. 

Wykorzystane technologie:
  - **Python**:
      - Flask,
      - Flask Mail,
      - Flask MySQL,
      - Jinja2,
      - XlsxWriter,
  - **SQL**
  - **HTML**
  - **CSS**:
    - Bootstrap,
    - Bootstrap Notify,
  - **JavaScript**:
    - Ajax,
    - Particles.js.
